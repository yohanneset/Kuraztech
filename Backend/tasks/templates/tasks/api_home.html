Task Manager API is Working

You can use the following endpoints to interact with the API:

  • GET /api/tasks/ — List all tasks
  • POST /api/tasks/ — Create a new task
  • GET /api/tasks/<id>/ — Retrieve a task
  • PUT /api/tasks/<id>/ — Update a task
  • PATCH /api/tasks/<id>/ — Partially update a task
  • DELETE /api/tasks/<id>/ — Delete a task
  • GET /api/tasks/?status=finished — Filter finished tasks
  • GET /api/tasks/?status=pending — Filter pending tasks

Use an API tool like Postman, Insomnia, or cURL to test the endpoints.
